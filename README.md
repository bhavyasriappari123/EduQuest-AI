# EduQuest AI

Smart Question Paper Generator Using RAG and Agentic AI

## Features

- Generate MCQs from notes
- Generate 10-mark questions
- Create study plans
- Generate summaries
- AI-powered learning assistance

## Tech Stack

- Frontend: HTML
- Backend: Flask
- AI Model: Google Gemini
- Language: Python

## Project Architecture

User Notes
    ↓
RAG Retrieval
    ↓
Gemini AI
    ↓
Question Generation
    ↓
Study Plan Creation

## Installation

pip install -r requirements.txt

python app.py
